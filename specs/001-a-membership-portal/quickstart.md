# Quickstart

1. Authenticate
   - Start login, complete authorization, and establish a session.
2. Submit a nomination
   - Open Nomination page → fill form → submit → see confirmation.
3. Browse members
   - Open Members page → search → open a member profile.
4. Calendar CRUD
   - Create an event → edit it → delete it; observe permission enforcement.
5. Moderation
   - Report a post as a member → action appears in moderation queue → admin hides/removes → author notified.
