🏢 **Professional WordPress Templates for Chamber of Commerce Websites**

A mobile-responsive template system designed specifically for the Richfield Area Chamber of Commerce (RACC) and adaptable for any chamber or business organization.

## 🎯 **What's Included:**

### **✅ One Template:**
- **`racc-template.json`** - Everything you need in one file
- **`racc-template-styles.css`** - CSS styles for all functionality
- **`racc-template-script.js`** - JavaScript for all interactive features
- Professional header with logo and navigation
- Content area for your custom content
- Professional footer with links and contact info
- Mobile-responsive slide-out menu
- Light/dark theme toggle functionality

### **📚 Documentation:**
- **`SIMPLE-INSTALLATION.md`** - Easy 3-step installation guide
- **`LOGO-CUSTOMIZATION-GUIDE.md`** - How to upload and customize logos

## 🚀 **Quick Start:**

1. **Download:** `racc-template.json`
2. **Import:** Divi > Divi Library > Import & Export
3. **Apply:** Load template on any page
4. **Add CSS/JS:** Copy styles to Additional CSS and add script
5. **Connect:** Link your WordPress menus
6. **Customize:** Add your content and upload your logo

**👉 See `INSTALLATION.md` for detailed instructions**

## ✨ **Features:**

- **🎨 Professional Design** - Clean, modern chamber-appropriate styling
- **📱 Mobile Responsive** - Perfect on all devices and screen sizes
- **🔗 WordPress Integration** - Connects seamlessly with your existing menus
- **⚡ Easy Customization** - Upload logos and edit content through DIVI Builder
- **🌙 Theme Toggle** - Light/dark mode switching with local storage
- **✨ Interactive Features** - Smooth scrolling, hover animations, mobile menu
- **♿ Accessible** - Built with accessibility best practices
- **🎯 Chamber-Focused** - Designed specifically for chamber organizations

## 🎨 **Customization:**

- **Logo:** Click image modules to upload your organization's logo
- **Colors:** Modify through DIVI Builder design settings
- **Content:** Replace example content with your own using any DIVI modules
- **Footer:** Edit contact info and links in footer code modules
- **Navigation:** Connect your existing WordPress menus

## 📋 **Requirements:**

- WordPress with DIVI theme (4.27.4+ recommended)
- Admin access to WordPress dashboard
- Basic familiarity with DIVI Builder

## 🤝 **Support:**

This template system is designed to be simple and self-contained. All styling and functionality is embedded in the template, so no additional plugins or theme modifications are required.
