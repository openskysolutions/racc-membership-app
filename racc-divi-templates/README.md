# RACC DIVI Templates
🏢 **Professional WordPress Templates for Chamber of Commerce Websites**

**⚡ LATEST UPDATE**: Fixed header/footer layout and color system to match React app exactly. Template now includes proper 2-row header design, corrected footer grid layout, and accurate color variables for consistent branding.

A mobile-responsive template system designed specifically for the Richfield Area Chamber of Commerce (RACC) and adaptable for any chamber or business organization.

## 🎯 **What's Included:**

### **✅ One Complete Template:**
- **`racc-template.json`** - Everything you need in one self-contained file
- **Embedded CSS** - All styles included directly in the template
- **Embedded JavaScript** - All interactive functionality built-in
- Professional header with logo and navigation
- Content area for your custom content
- Professional footer with links and contact info
- Mobile-responsive slide-out menu
- Light/dark theme toggle functionality

### **📚 Documentation:**
- **`INSTALLATION.md`** - Easy 2-step installation guide
- **`LOGO-CUSTOMIZATION-GUIDE.md`** - How to upload and customize logos
- **`PACKAGE-INFO.md`** - Technical details and customization options

## 🚀 **Quick Start:**

1. **Download:** `racc-template.json`
2. **Import:** Divi > Divi Library > Import & Export
3. **Apply:** Load template on any page
4. **Connect:** Link your WordPress menus
5. **Customize:** Add your content and upload your logo

**👉 See `INSTALLATION.md` for detailed instructions**

- **Logo:** Click image modules to upload your organization's logo

## 📋 **Requirements:**

- WordPress with DIVI theme (4.27.4+ recommended)
- Admin access to WordPress dashboard
- Basic familiarity with DIVI Builder

This template system is designed to be simple and self-contained. All styling and functionality is embedded directly in the template, so no additional plugins, theme modifications, or external files are required.
