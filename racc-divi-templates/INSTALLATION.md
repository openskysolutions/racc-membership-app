# RACC Template - Installation

## 🎯 **What This Is:**

One complete RACC template that includes:

✅ **Header** with logo and navigation  
✅ **Content area** where you add your page content  
✅ **Footer** with links and contact info  
✅ **Mobile menu** that slides out on mobile devices  
✅ **Built-in JavaScript** for all interactive functionality

## 📥 **Installation (3 Simple Steps):**

### **Step 1: Import the Template**

1. **Download:** `racc-template.json`
2. **WordPress Admin > Divi > Divi Library > Import & Export**
3. **Choose File:** Select `racc-template.json`
4. **Click:** "Import Divi Builder Layouts"
5. **Verify:** Template appears in Divi Library as "RACC Page Template"

### **Step 2: Connect Your Menus**

1. **Create a new page** or edit existing page
2. **Use Divi Builder** and **Load from Library**
3. **Select:** "RACC Page Template"
4. **Find the header menu module** and click it
5. **Menu Settings:** Select "Top Menu (Primary Menu)"
6. **Find the mobile menu module** and click it
7. **Menu Settings:** Select "Mobile Menu"
8. **Save the page**

### **Step 3: Add Required CSS**

**IMPORTANT:** The template needs additional CSS to work properly.

1. **Go to:** WordPress Admin > Appearance > Customize > Additional CSS
2. **Copy and paste** the entire contents of `racc-template-styles.css` 
3. **Save** the customizer changes

**Note:** The JavaScript functionality is already embedded in the template, so you don't need to add any additional scripts!

### **Step 4: Customize Your Content**

1. **Delete the example content** in the middle section
2. **Add your own content** using any Divi modules you want:
   - Text modules
   - Images 
   - Buttons
   - Contact forms
   - Galleries
   - Whatever you need!
3. **Update footer links** if needed (click the code modules in footer)
4. **Upload your logo** (click the logo image modules)

## ✅ **That's It!**

You now have a complete RACC page with:

- Professional header and footer
- Your WordPress menus connected
- **Working mobile menu with slide-out functionality**
- **Built-in JavaScript for all interactive features**
- Mobile-responsive design
- Your custom content in the middle

## 🔄 **For Additional Pages:**

1. **Create new page**
2. **Load from Library > "RACC Page Template"**
3. **Replace the middle content** with your page-specific content
4. **Menus and header/footer stay consistent** across all pages

**Note:** You only need to add the CSS once (in Additional CSS), and it will apply to all pages using the template.

## 🎨 **Customization:**

- **Logo:** Click image modules to upload your logo
- **Colors:** Modify in Divi Builder design settings or edit the CSS
- **Footer Links:** Edit the code modules in footer columns
- **Contact Info:** Update footer contact details
